订阅协议

最后更新时间：2025年7月

订阅说明

2. 订阅购买与支付

2.1 购买流程

  1. 在应用内点击"升级到高级版"
  2. 选择年度订阅方案
  3. 通过Apple ID确认购买
  4. 使用Face ID、Touch ID或密码验证
  5. 订阅立即生效，开始享受高级功能

2.2 支付方式

  • 所有订阅通过Apple App Store处理
  • 支持Apple ID绑定的所有支付方式
  • 包括信用卡、借记卡、Apple Pay、礼品卡等
  • 费用将从您的Apple ID账户扣除

2.3 价格说明

  • 中国区用户：¥128/年（含税）
  • 美国区用户：$19.9/年（不含税，税费由Apple处理）
  • 其他地区：价格由Apple根据汇率自动换算
  • 价格显示：购买前会显示确切的本地价格

3. 自动续费

重要提醒：

  • 订阅将自动续费，除非您主动取消
  • 续费将在当前订阅期结束前24小时内进行
  • 续费费用将从您的Apple ID账户自动扣除

3.1 续费机制

  • 订阅期结束前24小时自动续费
  • 续费成功后订阅期延长12个月
  • 续费价格按当时的订阅价格计算
  • 续费失败会收到通知，有24小时宽限期

3.2 续费通知

  • 续费前会收到Apple的提醒通知
  • 续费成功后会收到确认邮件
  • 可在Apple ID设置中查看订阅状态

4. 取消订阅

4.1 取消方法

您可以通过以下方式取消订阅：

  1. 打开iOS设置应用
  2. 点击您的Apple ID（顶部个人信息）
  3. 选择"订阅"
  4. 找到"习行ActHab"并点击
  5. 选择"取消订阅"并确认

或者通过App Store应用：

  1. 打开App Store应用
  2. 点击右上角个人头像
  3. 选择"订阅"
  4. 找到"习行ActHab"并管理订阅
  5. 选择"取消订阅"

4.2 取消时间要求

重要提醒：

  • 必须在当前订阅期结束前至少24小时取消订阅
  • 取消后订阅将在当前期间结束时停止
  • 取消订阅后您仍可使用高级功能直到当前期间结束

4.3 取消后果

  • 当前订阅期结束后失去高级功能访问权限
  • 数据仍保留在设备上，不会丢失
  • 可以随时重新订阅恢复高级功能
  • 重新订阅时可能需要重新配置某些设置

5. 退款政策

5.1 退款渠道

所有退款请求需要通过Apple App Store处理：

  • 访问 reportaproblem.apple.com
  • 使用您的Apple ID登录
  • 找到习行ActHab的订阅购买记录
  • 选择"申请退款"并说明原因

5.2 退款条件

退款申请需要符合Apple的退款政策：

  • 技术问题导致无法正常使用
  • 意外购买或未授权购买
  • 应用功能与描述严重不符
  • 其他符合Apple退款政策的情况

5.3 退款处理

  • 处理时间：通常3-5个工作日
  • 退款方式：原支付方式退回
  • 退款影响：退款批准后立即失去高级功能访问权限
  • 数据保留：本地数据不会因退款而删除

6. 订阅管理

6.1 查看订阅状态

您可以通过以下方式查看订阅状态：

  • 应用内：设置 → 订阅状态
  • iOS设置：设置 → Apple ID → 订阅
  • App Store：个人资料 → 订阅

6.2 订阅信息包含

  • 当前订阅状态（活跃/已取消/已过期）
  • 订阅开始和结束日期
  • 下次续费日期和金额
  • 订阅历史记录

6.3 多设备同步

  • 订阅状态在所有使用相同Apple ID的设备上同步
  • 在新设备上登录相同Apple ID即可恢复订阅
  • 不支持跨Apple ID共享订阅

7. 价格变更

7.1 价格调整通知

如果订阅价格发生变更：

  • 我们会提前至少30天通知您
  • 通知方式包括应用内通知、邮件等
  • 现有订阅在当前期间内价格不变
  • 续费时将按新价格收费

7.2 价格变更权利

  • 您可以在价格变更前取消订阅
  • 取消后可在当前期间结束前继续使用高级功能
  • 如不取消，续费时将自动按新价格收费

8. 服务可用性

8.1 服务保障

  • 我们保证订阅服务的持续可用性
  • 如遇技术问题，我们会尽快修复
  • 重大服务中断时，我们会相应延长订阅期
  • 服务永久停止时，我们会提前通知并处理退款

8.2 维护通知

  • 计划维护会提前通知用户
  • 紧急维护会尽快发布通知
  • 维护期间可能影响部分功能使用
